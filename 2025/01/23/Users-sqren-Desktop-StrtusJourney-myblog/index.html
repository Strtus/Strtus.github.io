    • 0.前言

    Hexo 是一款基于 Node.js 的静态博客框架，能够轻松生成静态网页，并将网站内容预先渲染为 HTML 文件，从而避免了服务器实时渲染的需求。这与传统的动态博客不同，后者依赖于数据库和服务器端脚本来动态生成内容。因此，静态博客具备更快的加载速度、较低的服务器资源消耗和更高的安全性。由于内容以静态文件形式存储和分发，静态博客可以通过如 GitHub Pages 等托管服务实现免费部署，降低搭建成本。此外，Hexo 对 Markdown 和 LaTeX 的支持非常友好，渲染速度快，尤其适合用于构建技术型或知识分享类博客。
